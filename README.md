# Rubber ducky with cjmcu bad usb
##### Description: This is where you will find the firmware for the cjmcu bad usb atmega32u4 to code it like a rubber ducky and use it directly from the sd card.
##### Objective: The aim of this project is basically; be able to run duckyscripts v3.0 or newer directly from your sd card without flashing the firmware of the board and do it only one time.

# How to install it?
#### 1. You must download the sketch, change some settings like the layout you want to use and upload it to your board.
#### 2. Then you must have a txt file in your sd card and it should work with no problems.
